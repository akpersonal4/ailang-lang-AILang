# If/Else

A minimal example demonstrating conditional control flow with if/else.

## Concepts Demonstrated

- Variable declaration with `let`
- If/else conditional branching
- Comparison operators
- Return values

## How to Run

```bash
ail run main.ail
```

## What to Try

- Add an `else` branch that returns a different value
- Chain multiple conditions to check ranges (e.g., x > 5, x > 10, x > 20)

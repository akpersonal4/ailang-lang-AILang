# Mini Rule Engine

Evaluates business rules for premium discounts and free shipping using compound conditions with AND/OR logic.

## Concepts Demonstrated

- Compound boolean conditions (nested if for AND, separate checks for OR)
- Rule evaluation pattern
- Multi-rule composition in main
- Conditional output based on rule results

## How to Run

```bash
ail run main.ail
```

## What to Try

- Add a new rule (e.g., loyalty points eligibility) with its own conditions
- Modify the age or membership thresholds in existing rules to change eligibility

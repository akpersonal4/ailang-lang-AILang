# Patterns

A collection of reusable algorithmic patterns for AILang, including recursive data processing, string manipulation, and graph algorithms.

## Concepts Demonstrated

- Recursive filter, map, and reduce over lists
- Manual string find and split implementations
- Graph construction and traversal (dependency graph, topological sort)
- CSV and JSON data store patterns with import modules

## How to Run

```bash
ail run <pattern_name>.ail
```

## What to Try

- Run `recursive_reduce.ail` to see sum and product accumulation
- Run `topological_sort.ail` to observe dependency ordering output
- Modify the sample graph in `dependency_graph.ail` to add new nodes and edges

# Invoice Generator

Generates invoices with line items, subtotal, GST calculation, payment tracking, and due amount computation.

## Concepts Demonstrated

- Arithmetic with percentage calculations (GST)
- Nested function calls for line item totals
- Payment status logic with if/else
- Multi-step financial computation

## How to Run

```bash
ail run main.ail
```

## What to Try

- Add more line items to the invoice
- Change the GST rate from 18% to another value and update the total

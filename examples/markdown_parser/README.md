# Markdown Parser

Classifies markdown lines into headings, list items, or paragraphs using exact string matching.

## Concepts Demonstrated

- String equality comparison for classification
- Multi-class classification with if/else
- Function composition for line processing
- Print-based output for classification results

## How to Run

```bash
ail run main.ail
```

## What to Try

- Add more markdown patterns (e.g., blockquotes, code blocks) to the classifier
- Add a function that counts how many of each line type appear in the input

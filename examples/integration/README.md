# Integration

A minimal module demonstrating a single utility function (`max`) without a `main()` entry point, designed for import by other modules.

## Concepts Demonstrated

- Module-level function definitions
- Conditional return values
- Reusable utility functions

## How to Run

```bash
ail build max_function.ail
```

## What to Try

- Add `min` and `clamp` utility functions to the module
- Import this module from another AILang file and call `max()`

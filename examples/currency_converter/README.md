# Currency Converter

Converts between USD and multiple currencies using fixed exchange rates, including round-trip conversion.

## Concepts Demonstrated

- Multi-branch if/else for currency dispatch
- Arithmetic with integer division
- Function composition and chaining
- String comparison for routing

## How to Run

```bash
ail run main.ail
```

## What to Try

- Add more currencies (e.g., CAD, AUD) to the `convert_usd()` function
- Add reverse conversion functions (EUR to USD, GBP to USD)

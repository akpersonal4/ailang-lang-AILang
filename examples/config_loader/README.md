# Config Loader

Demonstrates loading configuration values through dedicated getter functions (simulating a config file loader).

## Concepts Demonstrated

- Functions returning constant values
- Variable assignment from function returns
- Print output formatting

## How to Run

```bash
ail run main.ail
```

## What to Try

- Add more configuration keys (e.g., `get_timeout()`, `get_max_connections()`)
- Add validation functions to check config values are within valid ranges

# Text Search Tool

Searches a fixed corpus of text lines for exact keyword matches and counts occurrences.

## Concepts Demonstrated

- Exact string equality for matching
- Accumulator pattern for match counting
- Multi-line corpus search
- Conditional output on match

## How to Run

```bash
ail run main.ail
```

## What to Try

- Add more lines to the corpus and search for different keywords
- Add a function to search for partial matches (substring containment)

# CSV Reader

Simulates parsing a CSV row by extracting individual fields through dedicated accessor functions.

## Concepts Demonstrated

- Functions returning different types (string, integer)
- Parameterized print function
- Function composition

## How to Run

```bash
ail run main.ail
```

## What to Try

- Add functions to handle multiple CSV rows
- Add a validation function to check field types (age must be a number)

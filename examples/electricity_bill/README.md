# Electricity Bill Calculator

Calculates electricity bills using slab-based pricing with fixed charges and surcharge.

## Concepts Demonstrated

- Slab-based pricing with cascading if/else
- Multi-step financial computation
- Function decomposition for billing components
- Arithmetic with percentage calculations

## How to Run

```bash
ail run main.ail
```

## What to Try

- Change the `current_reading` value to compute bills for different consumption levels
- Add a new slab tier (e.g., 700+ units at 12 per unit) to the `energy_charge()` function

# Expression Evaluator

Evaluates simple postfix expressions with basic arithmetic operators and division-by-zero handling.

## Concepts Demonstrated

- String comparison for operator dispatch
- Division by zero error handling
- Function decomposition for evaluation steps
- Arithmetic operator selection via if/else

## How to Run

```bash
ail run main.ail
```

## What to Try

- Add the modulo (`%`) operator to `eval_binary()`
- Extend `eval_postfix_three_tokens()` to handle subtraction and multiplication examples

# Shopping Cart Calculator

Computes shopping cart totals with item quantities, discount, tax, and free shipping threshold.

## Concepts Demonstrated

- Nested function calls for line totals
- Percentage-based discount and tax application
- Threshold-based shipping logic
- Multi-step total computation

## How to Run

```bash
ail run main.ail
```

## What to Try

- Change the `free_shipping_threshold` to see shipping cost change
- Add a coupon code function that applies a fixed discount amount

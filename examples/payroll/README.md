# Payroll Calculator

Calculates employee payroll with basic pay, HRA, special allowance, PF, professional tax, and income tax deductions.

## Concepts Demonstrated

- Multi-component salary breakdown
- Percentage-based deductions (PF, tax slabs)
- Nested function calls for gross-to-net computation
- Tax slab logic with cascading if/else

## How to Run

```bash
ail run main.ail
```

## What to Try

- Change the CTC amount to compute payroll for different salary levels
- Add a bonus component to the salary breakdown

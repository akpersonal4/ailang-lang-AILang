# Prime Checker

Checks whether a given integer is a prime number using basic divisibility rules.

## Concepts Demonstrated

- Modulo operator for divisibility checking
- Early return pattern
- Multiple base cases

## How to Run

```bash
ail run main.ail
```

## What to Try

- Change the argument to `is_prime()` to test different numbers
- Add a function that finds all primes up to N using recursion

# Attendance Calculator

Calculates attendance percentage and determines eligibility based on a 75% threshold, including how many more classes are needed.

## Concepts Demonstrated

- Function composition
- Arithmetic with percentages
- Conditional logic for eligibility
- Mathematical formula computation

## How to Run

```bash
ail run main.ail
```

## What to Try

- Change `total` and `attended` values to see different eligibility outcomes
- Modify the `minimum_required()` function to change the attendance threshold

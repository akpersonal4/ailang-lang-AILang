# Loan EMI Calculator

Calculates loan EMI using simple interest approximation, total payment, interest paid, and eligibility check.

## Concepts Demonstrated

- Multi-step financial computation
- Percentage-based eligibility threshold
- Nested function calls
- Arithmetic with division

## How to Run

```bash
ail run main.ail
```

## What to Try

- Change the `principal` and `rate` values to compute EMI for different loan amounts
- Modify the `loan_eligibility()` threshold from 40% to another percentage

# Functions

Demonstrates basic function definitions, calls, and combining return values with arithmetic.

## Concepts Demonstrated

- Function definitions with parameters
- Return values
- Combining function results with operators

## How to Run

```bash
ail run main.ail
```

## What to Try

- Add more math functions (e.g., `abs`, `min`, `max`) and call them from `main()`
- Create a function that composes `square` and `cube` together

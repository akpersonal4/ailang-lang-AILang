# Password Validation

Validates passwords against length, digit, and special character requirements.

## Concepts Demonstrated

- Nested if/else for multi-condition validation
- Boolean flag composition
- Input validation pattern
- Conditional output (VALID/INVALID)

## How to Run

```bash
ail run main.ail
```

## What to Try

- Add an uppercase letter requirement to the `validate_password()` function
- Test additional password cases with different flag combinations

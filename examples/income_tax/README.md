# Income Tax Calculator

Calculates income tax using slab-based rates with standard deduction and education cess.

## Concepts Demonstrated

- Slab-based tax computation with cascading if/else
- Percentage calculations for cess
- Function decomposition for tax components
- Multi-step financial calculations

## How to Run

```bash
ail run main.ail
```

## What to Try

- Change the `gross` income amount to compute tax for different income levels
- Add a new tax slab (e.g., 2000001+ at 30%) to the `income_tax()` function

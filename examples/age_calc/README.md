# Age Calculator

Calculates age from birth year and determines generation, senior status, and years to retirement.

## Concepts Demonstrated

- Function definitions and calls
- Arithmetic operations
- Conditional if/else logic
- Return values from functions

## How to Run

```bash
ail run main.ail
```

## What to Try

- Change the `birth_year` variable to calculate age for different people
- Modify the `generation()` function thresholds to test different generational boundaries

# Student Records

Manages student records with grade calculation, pass/fail determination, and average computation.

## Concepts Demonstrated

- Letter grade mapping with cascading if/else
- Average computation from multiple scores
- Pass/fail threshold logic
- Function decomposition for record processing

## How to Run

```bash
ail run main.ail
```

## What to Try

- Add more students with different score combinations
- Add a function to find the highest-scoring student among the records

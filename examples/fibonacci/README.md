# Fibonacci

Generates the nth Fibonacci number using recursion.

## Concepts Demonstrated

- Recursive function calls
- Base case handling (n=0, n=1)
- Return values from recursive functions

## How to Run

```bash
ail run main.ail
```

## What to Try

- Change the argument to `fibonacci()` to compute different values in the sequence
- Add a function that prints the first N Fibonacci numbers (using recursion instead of loops)

# Date Validation

Validates calendar dates including leap year detection and per-month day limits.

## Concepts Demonstrated

- Nested if/else logic
- Leap year algorithm with multiple conditions
- Modular arithmetic
- Function with side-effect-free validation

## How to Run

```bash
ail run main.ail
```

## What to Try

- Add validation for month=13 or day=0 edge cases
- Add a function that calculates the day-of-year from a date

# BMI Calculator

Computes Body Mass Index from weight and height, categorizes the result, and provides health advice.

## Concepts Demonstrated

- Arithmetic with integer math (avoiding floats)
- Multi-branch if/else for categorization
- Function decomposition
- String return values from conditions

## How to Run

```bash
ail run main.ail
```

## What to Try

- Change the weight and height values to see different BMI categories
- Add additional health advice categories to the `health_advice()` function

# File Copy Utility

Validates pre-conditions for a file copy operation (path checks, size validation) without actual I/O.

## Concepts Demonstrated

- Input validation with if/else
- Multiple guard conditions
- String equality comparison
- Error reporting via print

## How to Run

```bash
ail run main.ail
```

## What to Try

- Pass empty strings as source or dest to trigger error messages
- Add a function to validate file extensions (e.g., only copy `.txt` files)

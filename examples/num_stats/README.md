# Number Statistics

Computes min, max, and average from three hardcoded numbers using comparison-based logic.

## Concepts Demonstrated

- Comparison operators for min/max
- Arithmetic for average computation
- Variable reassignment within conditions
- Function decomposition

## How to Run

```bash
ail run main.ail
```

## What to Try

- Add a `get_median()` function for three values
- Add a function to compute the range (max - min)

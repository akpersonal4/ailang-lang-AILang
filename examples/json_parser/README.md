# JSON Parser

Simulates field-level JSON parsing with validation for name and age fields.

## Concepts Demonstrated

- Functions returning typed values (string, integer, boolean-as-int)
- Field validation with if/else
- Conditional output based on validation results
- String equality checks

## How to Run

```bash
ail run main.ail
```

## What to Try

- Add validation for additional fields (e.g., email format, age range)
- Add a function to check if required fields are present

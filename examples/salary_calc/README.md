# Employee Salary Calculator

Computes gross salary from base with HRA and DA, applies PF and tax deductions, and determines salary band.

## Concepts Demonstrated

- Percentage-based salary components
- Tax slab computation with cascading if/else
- Nested function calls for deductions
- String-based salary band classification

## How to Run

```bash
ail run main.ail
```

## What to Try

- Change the base salary to see how deductions and net pay change
- Modify the HRA and DA percentages in `gross_salary()` calls

# Directory Tree Printer

Prints a hardcoded directory tree structure with indentation to simulate filesystem traversal output.

## Concepts Demonstrated

- Depth-based branching logic
- Conditional formatting with if/else
- Print-based output formatting

## How to Run

```bash
ail run main.ail
```

## What to Try

- Add more directories and files to the tree output
- Modify the `print_node()` function to handle deeper nesting levels

# Library Book Manager

Manages library book operations: fine calculation, availability status, borrow eligibility, and renewal limits.

## Concepts Demonstrated

- Financial computation (per-day fine rates)
- Boolean flag logic with integer returns
- Threshold-based eligibility checks
- Function composition

## How to Run

```bash
ail run main.ail
```

## What to Try

- Change `days_late` to see how fine amounts change
- Add a function to calculate maximum allowed borrowing days

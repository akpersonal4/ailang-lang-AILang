# Voting Eligibility Checker

Checks voting eligibility based on citizenship, age (18+), and criminal record status.

## Concepts Demonstrated

- Compound conditions with nested if/else (AND logic)
- Boolean flag composition
- Multi-condition eligibility checking
- Conditional output for eligible/ineligible

## How to Run

```bash
ail run main.ail
```

## What to Try

- Add a residency duration requirement to the eligibility check
- Test edge cases like age=18 (boundary) or missing citizenship ID

# Simple Banking System

Simulates basic banking operations: deposit, withdrawal with balance checks, and balance tier display.

## Concepts Demonstrated

- Function return values for state updates
- Input validation with if/else
- Error handling via print messages
- State mutation through reassignment

## How to Run

```bash
ail run main.ail
```

## What to Try

- Try depositing a negative amount to see the error handling
- Change the initial balance to test different balance tier displays (HIGH/MEDIUM/LOW)

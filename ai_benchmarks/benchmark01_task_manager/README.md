# Benchmark 01 – Personal Task Manager

This benchmark tests the ability of an AI model to generate a complete, working Personal Task Manager application in AILang.

The application implements CRUD operations, task status management, search, statistics, JSON persistence, and CSV export.

The benchmark is evaluated against the following criteria:

1. Correct AILang syntax and standard library usage.
2. Successful compilation (`ail build`).
3. Successful execution (`ail run`).
4. Correct output matching the expected behaviour.
5. Code quality and modularity.

The benchmark is run with the following models:
- Gemini 2.5 Pro
- GPT‑4o
- Claude 3.5 Sonnet
- Kimi
- DeepSeek

Each model’s output is stored in its own sub‑folder.
